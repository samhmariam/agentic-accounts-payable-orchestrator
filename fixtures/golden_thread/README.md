# Golden Thread Fixtures

These fixtures define the canonical Day 1-5 training scenario used by the new
lab scripts, notebooks, and deployment smoke tests.

- `package.json`: Day 1 intake package
- `candidate.json`: deterministic Day 1 extraction candidate
- `day3_invoice.json`: Day 3 retrieval invoice payload
- `day4_case.json`: Day 4 and Day 5 planning / approval case facts
