# Capstone Source of Truth

This folder is the canonical capstone contract for the AegisAP curriculum.

## Sequence

1. Day 10 foundation packet: build the bounded release packet and prove the production-readiness checkpoint.
2. Days 11-13 evidence accrual: add delegated identity, private networking, and integration-boundary proof without breaking the Day 10 release posture.
3. Day 14 final CAB defense: aggregate the Day 10-14 evidence into one final Capstone A packet and defend it in `cab_board` mode.
4. Capstone B transfer domain: apply the same Days 12-14 operating discipline to the claims-intake transfer domain for Top Talent evaluation.
