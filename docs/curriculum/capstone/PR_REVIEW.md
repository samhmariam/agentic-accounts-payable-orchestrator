# Capstone PR Review

This review happens before the Day 14 final CAB defense.

## Required Packet

- change summary
- risk assessment
- test evidence
- KQL replay evidence
- Revert Proof
- rollback plan
- release evidence
- one known limitation
- Day 10 foundation packet reference
- Day 14 final packet reference

## Review Contract

- each peer reviewer must ask at least 2 skeptical questions
- each peer reviewer must demand at least 1 concrete evidence artifact
- each peer reviewer must replay at least one saved KQL query
- each peer reviewer must inspect Revert Proof and challenge the recovery time box
- the board may require the learner to rerun the selected native proof live
- capstone peer roles are `client_ciso` and `infra_lead`, with the facilitator acting as CAB chair
