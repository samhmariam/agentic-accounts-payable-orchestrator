# AegisAP Pilot Measurement Plan

Run one pilot cohort before declaring the curriculum final. This converts the
program from “well designed” to “calibrated.”

## Pilot Scope

- One full cohort
- Recommended minimum sample size: 8 learners
- At least 2 trainers or reviewers should score the capstone so scoring drift
  is visible

## Daily Data To Collect

For each day, collect:

- start and end completion time
- artifact success on first attempt
- remediation count
- exit-ticket pass rate
- rubric averages by dimension

For checkpoint days, also collect:

- checkpoint pass on first attempt
- average recovery time if failed initially

## Capstone Data

- capstone pass rate
- common PR review findings
- average review response score
- average defense score
- top 5 failure causes

## Success Criteria For 95+ Readiness

- checkpoint first-pass rate >= 80%
- capstone pass rate >= 70% without rubric inflation
- no day with more than 35% of learners in remediation
- trainer scoring variance acceptable across at least 2 reviewers

## Post-Pilot Retro

Use `docs/curriculum/templates/PILOT_RETRO.md` to record:

- what ran long
- what learners misunderstood
- which artifacts failed most
- which day had the highest remediation burden
- what should be cut, tightened, or moved
