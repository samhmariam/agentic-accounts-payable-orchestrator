# AegisAP Graduate Profile

Bootcamp completion means the learner can perform the following independently on
bounded agentic-system tasks.

## Graduate

- add or modify a bounded policy rule safely
- debug a failing gate from artifact and release evidence
- translate a portal or telemetry finding into a permanent code or IaC change
- explain the runtime identity model and least-privilege posture
- instrument a bounded observability addition without exposing secrets or PII
- defend a cost-routing or safety tradeoff with evidence
- ship a bounded enhancement with tests, rollback, and release packet

## Strong FDE

- lead bounded architecture and review work inside this system with evidence
- defend identity, network, boundary, and release trade-offs to hostile reviewers
- turn portal or telemetry truth into durable repo or IaC changes without losing governance discipline

## Top Talent

- redesign and defend the end-to-end architecture alone for this bounded system
- transfer the same operating model into an unfamiliar domain without pre-loading
- run the final CAB, security, ops, and executive story without losing technical precision

## Not Yet Guaranteed

- own an open-ended multi-month system roadmap without support
- make major platform-level tradeoffs without review from senior engineering
- replace production release governance with solo judgment

Use this document to map bootcamp completion to staffing expectations, project
assignment, and follow-on mentoring.
